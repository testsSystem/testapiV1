# tech stack used

1. Nodejs/express.js
2. MySQL
3. Sequelize ORM
4. Sequelize Migrations

# Requirements

1. sequelize-cli
2. mysql
3. mysql2
4. npm i -g sequelize-cli mysql mysql2

# Getting started

1. npm i
2. sequelize db:create
3. sequelize db:migrate
4. npm run dev
